# EzoriTech

## Tecnología que vuelve a la vida

Sitio web oficial de EzoriTech, un centro especializado en reparación de celulares en Cúcuta.

### Servicios
- Cambio de pantallas
- Cambio de baterías
- Cambio de tapas
- Reparación de centros de carga
- Diagnóstico
- Microelectrónica
- Equipos con daño por humedad
- Reparaciones de software y hardware
